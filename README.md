# Canvas Clone Frontend

This is a react-based frontend for a Canvas-like project. The backend is located [here](https://github.com/dtmesa/cv-web-be). It lacks the full functionality of Canvas, but has many of the core features. A REST API is used to communicate with the backend. MongoDB is used for the database. Backend access is restricted by user sessions.

## Features

* User creation, sessions, roles, and permissions
* Assignment creation & editing
* Enrollment management
* Quiz creation and management
* User dashboard
* Canvas-inspired CSS
